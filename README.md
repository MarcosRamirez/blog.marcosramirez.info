# 🧑‍💻 Blog de Marcos Ramírez

Blog personal sobre tecnología, finanzas personales y productividad.

## 🚀 Stack

- [Jekyll](https://jekyllrb.com/) + [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy)
- Desplegado en [GitHub Pages](https://blog.marcosramirez.info)

## 🛠️ Desarrollo Local

```bash
bundle install
bundle exec jekyll serve
```

Accede en `http://localhost:4000`

## 📝 Publicar

Haz push a `master` → GitHub Actions compila y despliega automáticamente.

## 📂 Estructura

```
├── _posts/          # Artículos del blog
├── _includes/      # Componentes reutilizables
├── _layouts/       # Plantillas
├── assets/         # Imágenes, CSS, JS
├── _tabs/          # Páginas estáticas
└── _config.yml    # Configuración del sitio
```
